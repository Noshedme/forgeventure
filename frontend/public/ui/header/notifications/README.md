# Notification header assets

Place transparent PNG files here. Recommended size: 64x64 px. Minimum safe size: 48x48 px.

Required filenames:

- `notif-quest.png` - mission/quest notifications.
- `notif-medal.png` - achievement notifications.
- `notif-shop.png` - shop/new item notifications.
- `notif-shield.png` - streak, shield, protection notifications.
- `notif-message.png` - admin messages and system mail.
- `notif-empty.png` - empty inbox state.

Style notes:

- Use transparent background.
- Keep the main object centered with 6-8 px padding.
- Pixel-art or painted RPG icons both work, but keep contrast high.
- The UI applies class-colored glow automatically.
